Remove Null (ε) Productions

What are null productions?

A null production is any grammar rule that produces the empty string ε, e.g. A → ε. To remove them, we first find all nullable variables — variables that can derive ε directly or indirectly. Then, for every production containing a nullable variable, we create new versions with that variable both included and omitted.

What are unit productions?

A unit production is a rule of the form A → B, where both A and B are variables. We find every unit pair (A, B) — meaning A can reach B through unit productions — and replace those rules with all of B's non-unit productions.

What are useless symbols?

A symbol is useless if it can never contribute to generating a terminal string from the start symbol. This happens in two ways:

  • Non-generating: The variable can never produce a string of terminals, no matter how many steps.
  • Unreachable: The variable can never be reached from the start symbol — it's dead code.
