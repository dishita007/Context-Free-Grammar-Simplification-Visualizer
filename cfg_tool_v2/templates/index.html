Quick Reference
ε-productions

Rules like A → ε produce the empty string. Find all nullable variables, then create rules with/without them.

Unit productions

Rules like A → B map one variable to another. Replace by finding all unit pairs and substituting.

Useless symbols

A symbol is useless if it's non-generating (can't reach terminals) or unreachable from start symbol.

Format Guide
Variables

Uppercase letters: S, A, B

Terminals

Lowercase letters: a, b, c

Empty string

ε or λ for the empty/null production

What is CFG Simplification?

ε

Null Productions

A null production is any rule like A → ε. We remove these by generating equivalent rules that don't rely on the empty string.

A→B

Unit Productions

A unit production maps one variable directly to another: A → B. We replace these with the actual rules of B.

∅

Useless Symbols

A symbol is useless if it can never generate a terminal string, or if it can never be reached from the start symbol.

Did you know?
CNF & GNF

After simplification, grammars can be converted to Chomsky Normal Form (CNF) or Greibach Normal Form (GNF) for parsing algorithms like CYK.

Start symbol rule

If the start symbol S is nullable, the language contains ε. This is handled before removing ε-productions.

Step order matters
Why this order?

Null productions must be removed first, because removing unit productions could introduce new ε-derivations if done earlier.
